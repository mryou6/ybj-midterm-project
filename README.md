# 🧩 자료구조 놀이터

Stack, Queue, Binary Search Tree, Graph의 동작 과정을 직접 조작하고
시각적으로 체험할 수 있는 자료구조 학습 웹 애플리케이션입니다.

## 주요 기능

- Stack의 Push, Pop, Peek 체험
- Queue의 Enqueue, Dequeue, FRONT, REAR 체험
- Binary Search Tree의 삽입, 탐색, 순회 시각화
- Graph의 정점·간선 구성 및 DFS·BFS 단계별 탐색
- 결과 예측 활동과 학습 확인 문제

## 웹 애플리케이션의 역할

자료구조의 동작 과정을 직접 조작하고 시각적으로 확인할 수 있는
체험형 학습 환경을 제공합니다.

## 개발 목적

누구나 자료구조의 원리를 쉽고 친근하게 이해하도록 돕는 것을
목적으로 합니다.
